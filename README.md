# ChatGPT App Package

A package for developing apps using ChatGPT.

## Installation

```bash
npm install chatgpt-app-package
```

## Usage

```javascript
const { getChatGPTResponse } = require('chatgpt-app-package');

(async () => {
    const response = await getChatGPTResponse('Hello, ChatGPT!');
    console.log(response);
})();
```
